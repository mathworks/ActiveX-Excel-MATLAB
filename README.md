## Excel ファイルのセル幅自動調整([Link to doc](docs/CellAutoFit.md))

## Excel 内での計算結果を更新させる方法([Link to doc](docs/ExcelUpdates.md))

Excel シートに MATLAB から何らかの値を出力しその値に対しての Excel 内での処理を実行させます。
